Q2. Write the code to access element which is having tag as "h1"

ANS : let a=document.getElementsByTagName("h1") console.log(a) output: Elements having tag name as h1 will be stored in variable a in an array format/Element collection.

Q3. Write the code to access element which is having class as "box"

ANS : let a=document.getElementsByClassName("box") console.log(a) output: Elements having class name as box will be stored in variable a in an array format/Element collection.
